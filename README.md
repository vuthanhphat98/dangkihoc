## 🚀 About the Project

**Course Registration System** (dangkihoc) is a robust full-stack application designed to streamline the academic registration process. It provides an intuitive interface for students to browse, enroll in, and manage their classes, while offering administrators powerful tools to oversee course availability, capacities, and scheduling conflicts. 

---

## ✨ Key Features

* **User Authentication:** Secure and role-based login for students, instructors, and administrators.
* **Course Browsing:** Dynamic search and filter options to find available courses by department, schedule, or instructor.
* **Real-time Enrollment:** Seamless class registration with immediate conflict detection and capacity tracking.
* **Interactive Dashboard:** Personalized views displaying current schedules, academic progress, and instant notifications.
* **Admin Management:** Comprehensive administrative panel for creating courses, managing users, and generating system reports.

## 🏁 Getting Started

Follow these steps to set up the project locally for development and testing.

1. **Clone the repository:** `git clone https://github.com/vuthanhphat98/dangkihoc.git`
2. **Navigate to the directory:** `cd dangkihoc`
3. **Install dependencies:** Run `npm install`, `composer install`, or `flutter pub get` depending on the core environment.
4. **Environment setup:** Copy `.env.example` to `.env` and update your local database credentials.
5. **Database migration:** Execute the necessary SQL scripts or migration commands to set up your tables.
6. **Run the application:** Start your local development server to view the project.

---

## 📱 Usage

1. Launch the application and log in using your registered credentials.
2. Navigate to the **Courses** section to view the academic catalog.
3. Select your desired courses and click **Enroll** to add them to your schedule.
4. Check your **Dashboard** to verify your successful registrations and upcoming classes.
